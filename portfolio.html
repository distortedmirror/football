Distorted Mirror
{joking}I managed to break the laws of physics twice, first by bending light, and once again by modifying how light reflects.{/joking} =]

Open https://distortedmirror.github.io/cam.html with your Android or iPhone (and approve camera access) to see light bend!

Open https://distortedmirror.github.io/html/index.html on Android, iPhone or Desktop (and approve camera access) to see light reflect different than mirrors do.

Distorted Mirror stemmed from my endeavor to learn webGL for directly interfacing with Graphics Processing Units, which basically implements LA (Linear Algebra), VA (Vector Analysis) and PDE (Partial Differential Equations) mathematics in silicon.  I set a goal to implement a video texture map, after which I progressed to texture map video from a camera device.  I didn't even imagine it would implement a mirror, and visually realized it reflected differently than a mirror.  I have considered implementing it, per the laws of optics, with the proper physics equations.

WebGL Viewer
3D Web Apps are far more challenging than desktop 3D because you have to load and display it quickly, its not preloaded and can't have long loading and processing wait times.  WebGL Viewer is my attempt of loading ultra optimized content and quickly rendering the scene with 3 finger multi tap control and navigation.  The geometry was reduced, the texture maps were optimized for size and only the minimum was loaded.  One goal was to be able to scan objects from the real world, and was achieved.  Another goal was to be able to import any geometry from Google 3D warehouse, and was also successfully completed.  Objects in the scene, 3D controlling object selection, and scene building was implemented as well.  My favorite feature is the three finger tap control. Tap to accelerate forward, double tap to decelerate, two finger tap drag to rotate on all three axis and three finger tap drag to translate in all 3 axis - complete control with 3 fingers. You can test drive it at https://distortedmirror.github.io/football. A direct link for a 3D world of a monster in a football stadium can be viewed at https://tinyurl.com/WebGLFootballMonster. (Wait for a minute to load the scene.)
Web Builder
Web Builder brings a complete WYSIWYG HTML CSS Javascript development IDE to your browser, no installation required.  Although its not yet documented, it has a lot of features for advanced, rapid page design visually.  Features include drag and drop editor support of video, audio, image, markup and text files from external files for quick player and content creation.  Support for cut in paste of rich content from other pages or any other application.  In place visual content editing.  Drag and drop placement and sizing of elements. Tag creation from a list of all element types. Property editor and CSS styling editor.  
This portfolio page was created with Web Builder visually with literally zero code, and clean HTML was generated by Web Builder. This portfolio can be edited at https://distortedmirror.github.io/football/WebBuilder.html?js=portfolio.js.
Web Builder is at https://distortedmirror.github.io/football/WebBuilder.html.
Visual XML Designer
Visual XML Designer is a graphical drag and drop rubberbanding user interface that is essentially editing an XML file and is ensuring that the XML is validated to a DTD schema, a translated via XSLT via a XSL stylesheet to generate output and finally custom logic can be performed.  Visual Languages, Translator and Deployers enable visual designing that perform actions and generate output. The project is open source AWT Swing Java and can be found at  https://github.com/markbrito/vxd.
Job Intelligence
Job Intelligence is a web app that collects a job search profile and crawls dice.com for all the jobs that matches the search criteria, parses and extracts the job information, mines the data, generates advanced analytic charts, and was intended to submit a mass job application submission or otherwise help me find a job quickly.  The browsing path to crawl was a browser recording for test replay that was modified to scrape the data.  The database is SQL Server, the web portion was ASP.net and the charts were SQL Reporting Services.
USell
USell demonstrates a proof of concept where an app can be quickly created from a responsive web app by crawling the site and caching content for the app to display via a webview where all web requests are first checked to see if there is a local cache and if so, responds immediately from the local cache, and dynamic requests can also be responded to by local logic.  The result is an app that is generated from a website, but served locally from the app with lightning fast response time.  
You can take a look at the app at https://play.google.com/store/apps/details?id=com.seller.usell and the source can be viewed at https://github.com/distortedmirror/usell.  Note that the server component needs to be turned on for all server side functionality to work such as ordering, but the catalog works fine.
FlingPix
FlingPix enables for a picture to be taken and allow touch drag guestures to spin the image on several axis, depending on the differentials of the touch guesture. If your device has the magnetic gravity sensor, there is a mode where the image stays parallel to gravity no matter the orientation. Other modes bind the spin to the accelerometer or gyroscope. The source code is at https://github.com/markbrito/FlingPix.
