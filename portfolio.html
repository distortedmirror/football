Distorted Mirror
[joking]I managed to break the laws of physics twice, first by bending light, and once again by modifying how light reflects.[/joking] =]

Open https://distortedmirror.github.io/cam.html with your Android or iPhone (and approve camera access) to see light bend!

 Open https://distortedmirror.github.io/ on Android, iPhone or Desktop (and approve camera access) to see light reflect different than mirrors do.

Distorted Mirror stemmed from my endeavor to learn webGL for directly interfacing with Graphics Processing Units, which basically implements LA (Linear Algebra), VA (Vector Analysis) and PDE (Partial Differential Equations) mathematics in silicon, I set a goal to implement a video texture map, after which I progressed to texture map video from a camera device. I didn't even imagine it would implement a mirror, and visually realized it reflected differently than a mirror.  I have considered implementing it, per the laws of optics with the proper physics equations.

WebGL Viewer
3D Web Apps are far more challenging than desktop 3D because you have to load and display it quickly, its not preloaded and can't have long loading and processing wait times.  WebGL Viewer is my attempt of loading ultra optimized content and quickly rendering the scene with 3 finger multi tap control and navigation.  The geometry was reduced, the texture maps were optimized for size and only the minimum was loaded.  One goal was to be able to scan objects from the real world, and was achieved.  Another goal was to be able to import any geometry from Google 3D warehouse, and was also successfully completed.  Objects in the scene, 3d controlling object selection, and scene building was implemented as well.  My favorite feature is the three finger tap control. Tap to accelerate forward, double tap to decelerate, two finger tap drag to rotate on all three axis and three finger tap drag to translate in all 3 axis, complete control with 3 fingers. You can test drive it at https://distortedmirror.github.io/football.
Web Builder
Web Builder brings a complete WYSIWYG HTML CSS Javascript development IDE to your browser, no installation required.  Although its not yet documented,it has a lot of features for advanced, rapid page design visually.  Features include drag and drop editor support of video, audio, image, markup and text files from external files for quick player and content creation.  Support for cut in paste of rich content from other pages or any other application.  In place visual content editing.  Drag and drop placement and sizing of elements. Tag creation from a list of all element types. Property editor and CSS styling editor.  The IDE generates json that it uses to generate html.  Edit pages with almost no code, quickly with no coding. 
This portfolio page was created with Web Builder visually with literally zero code,and clean HTML was generated by Web Builder. This portfolio can be edited at https://distortedmirror.github.io/football/WebBuilder.html?js=portfolio.js
Web Builder is at https://distortedmirror.github.io/football/WebBuilder.html
